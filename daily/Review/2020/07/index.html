# 2020년 07월

# 7월을 시작하며

단국대학교에 입학해서 컴공을 전공하고 1,2,3 학년 1 학기까지 다니면서 무슨 배짱으로 공부를 그렇게 안했을까 후회가 많이 된다.

학교 공부 이외의 공부를 따로 한거도 아니고 그렇다고 학점을 잘 받은거도 아니다. (이번에 확인 해보니 누적 평균 3.5 ...)

대학생활 뿐만 아니라 지난 미국에서의 고등학교 생활, 연애 등등 항상 후회를 많이 한다.

사실 이런 무한 후회를 하며 과거가 바뀌길 바라는건 멍청한 행동이다. 사실 타임머신이 발명되지 않는한 불가능하다 (만약 발명된다 하더라도 쓰고 싶진 않다 ㅇㅅㅇ).

바꿀 수 있는건 미래인데 과거 백날 천날 후회해봤자 뭘하겠냐 👿

이번 7월 부터는 쓸데없는 생각에서 벗어나 내가 할 수 있는 공부, 연애에 최선을 다해서 살아봐야겠다. 남는게 별로 없더라도 내 스스로 열심히 살았다 만족할 수 있을 만큼만 해보자.

이번달 말에 10점 만점에 평가 해보자.

# 스터디

사실 약 3달전 스터디를 들어오기 전까지 과제 이외의 코딩을 해본적이 거의 없다. 즉, 백지 상태다.

일단 뭘 하든 기본이 중요하다 생각한다. 더 더욱 나는 아는게 없기 때문에 기본기부터 확실하게 하고 하나씩 공부 분야를 넓혀봐야 겠다. 이번달은 바닐라 JS, 알고리즘 공부 위주로 계획 중 이고 진행 중 이다.

# 바닐라 JS

  • 인사이드 자바스크립트
  • You Don't Know JS
  • 코어 자바스크립트

비록 많은 내용의 책이긴 하지만 내용이 많이 중복되고 충분히 읽을 수 있다. 이번달에 3권을 1회독 하고 다음달에 다시 한번더 보는 식으로 해봐야겠다.

평소 책 읽는 버릇이 없던터라 책보는게 쉽진 않지만 나름 내용이 재밌다.

# 알고리즘

매일 한 문제를 1 시간 정도 투자하고 풀이 성공하든 못하든 커밋을 하고 지난 코드를 리뷰하는 식으로 진행하는 중. 이대로만 매일매일 해야지.

# 7월을 마무리하며

7월은 정말 많은일이 일어난 한달 이였다. 좋은일도 있었고 나쁜일도 있었다.

# 스터디

자바스크립트 책 2권 (인사이드 자바스크립트, You Don't Know JS) 를 읽었고 코어 자바스크립트는 진행중이다.

알고리즘은 7월 말 기준으로 처음 프로그래머스 레벨 4문제에 도전하였다. 점점 문제 레벨을 올려가는게 마치 게임 캐릭터 키우는 기분이다.

일일커밋 은 여전히 잘 진행중. 알고리즘이나 블로그 업데이트용 커밋이 대부분이다 ㅋㅋ ,,, 알고리즘 문제 레벨을 올리는게 게임 캐릭터 레벨을 올리는 기분이라면 커밋은 롤 전적을 보는 느낌이 든다. 역시 컴공의 기본소양은 게임을 좋아하는것이 아닌가 싶다.

이번달 가장 큰 일은 인턴 합격이 아닌가 싶다. 아직 하나의 면접이 남았지만 결과가 어찌되든 열심히 인턴하면서 내가 하고 싶은 공부도 꾸준히 해야겠다. 앞으로 엄청 피곤해질듯 하다.

# 일상

점점 나이가 들어서 그런가 점점 혼자 다니는게 편하고 친구 만나는거 보단 집에서 혼자 노트북 보는게 더 편하고 좋다. 또 최근에는 인간관계로 인한 스트레스가 떄문인지 최근에는 집 - 스터디 카페 - 집 - 스터디 카페만 무한 반복중이다.

8월은 코딩 실력 향상의 달이다. 9월 인턴 시작하기 전에 여러가지 기초 지식 ( MVC 패턴, 디자인 패턴 등등 ) 더 집중해서 공부를 해봐야겠다. 파이팅 짜쓰 ~ 👊🏼

이번달 아주 주관적인 평점 : 3 / 5️ ⭐
