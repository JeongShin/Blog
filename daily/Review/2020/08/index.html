# 2020년 08월

# 8월을 시작하며

이번달은 앞으로 인턴 이후 4학년 학교 생활을 어떻게 지내고 졸업 후에 어떻게 취업을 준비할지 생각해 보았다.

일단 주변 모든 사람들이 강조하는 부분이 깊이 있는 공부 를 졸업 전까지 해보는게 중요할 듯 하다.

일단은 기본적인 웹 구현과 동작 방식을 이해하고 이를 기반으로 공부 범위를 넓혀가는게 중요할듯 하다.

웹을 공부하다 보니 만들어 보고 싶은 웹, 웹 서비스가 생각이 났다.

올해 하반기 계획

  1. 나의 여행 블로그 만들어보기

동적 웹으로 여행 후기를 올리는 블로그를 Rails 를 활용하여 만들어 보고 싶다.

인턴 마치고 겨울방학때 카메라 사서 혼자 여행을 떠나보자. 제주도든 어디든 떠나서 풍경 사진을 찍고 블로그에 업로드 해보자.

  1. 꾸준히 알고리즘 풀이

알고리즘 풀이는 지금 하는대로만 꾸준히 이어가면 될듯하다. 게을리 하지말자.

비동기, 데이터 통신, 객체지향과 함수형 프로그래밍, 렌더링 처리
